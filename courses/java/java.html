Introduction to Java

Java is a popular, general-purpose, object-oriented programming language developed by Sun Microsystems (now owned by Oracle) in the 1990s.

Java is widely used in a variety of applications, including web development (e.g., Java servlets, JavaServer Pages), mobile applications (Android apps), enterprise software, scientific computing, and game development.

Features of java

Object-Oriented: Java follows the principles of object-oriented programming (OOP), which includes concepts like classes, objects, inheritance, polymorphism, and encapsulation.

Platform-Independent: Java programs are compiled into bytecode, which can be executed by the JVM on any platform that supports it, making Java code highly portable across different operating systems and hardware architectures.

Automatic Memory Management: Java employs automatic memory management through a garbage collector, which automatically reclaims memory occupied by objects that are no longer in use, relieving developers from the task of manual memory management.

How java works?

Java is compiled into the bytecode and then it is interpreted to machine code.

Basic structure of java

From above:

public class Main{ : This line declares a new class named "Main" and makes it accessible from anywhere (public).

public static void main(String[] args){ : This is the main method, which is the entry point of a Java program. It must be present for the program to run.

  • public: This keyword means the method can be accessed from anywhere.
  • static: This keyword indicates that the method belongs to the class itself, not any specific instance (object) of the class.
  • void: This keyword specifies that the method does not return any value.
  • main: This is the name of the method, which must be "main" for the program to run.
  • (String[] args): This part defines the input parameters of the method, which can be used to pass command-line arguments to the program.

Chapter 1 - Variables and datatypes

Variable

A variable is a named container that stores a value of a specific data type.


Example:

From above:

int x =5;

  • int is a data type that represents integer values (whole numbers).
  • xis the variable name.
  • 5 is the value assigned to the variable x.

double y = 3.5;

  • double is a data type that represents floating-point numbers (numbers with decimal points).
  • y is the variable name.
  • 3.5 is the value assigned to the variable y.

String name = "john";

  • String is a data type that represents text or sequence of characters.
  • name is the variable name.
  • "john"is the string value assigned to the variable name. Note that strings are enclosed in double quotes.

boolean z = true;

  • boolean is a data type that can only hold one of two values: true or false.
  • z is the variable name.
  • true is the boolean value assigned to the variable z.

Parameter and Arguments

- Parameters are the variables defined in the method declaration.

- Arguments are the actual values that are passed to the method when it is called.

In the add method, int a and int b are parameters. When the method is called with calculator.add(5, 3), 5 and 3 are the arguments that get assigned to the a and b parameters, respectively.
